# Contributing

## Prerequisites

- Node.js 20+
- pnpm 9+

## Setup

```bash
pnpm install
pnpm dev
```

## Quality Checks

```bash
pnpm lint
pnpm typecheck
pnpm build
```

## Commit Guidelines

- Gunakan commit message yang jelas.
- Fokus satu perubahan logis per commit.
- Tambahkan/update docs jika API berubah.

## Pull Request

- Jelaskan masalah dan solusi.
- Sertakan screenshot jika ada perubahan UI.
- Pastikan seluruh command quality checks lolos.
