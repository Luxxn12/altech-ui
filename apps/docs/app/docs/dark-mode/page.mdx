# Dark Mode

Altech UI supports dark mode out of the box using token overrides under `.dark` or `[data-theme="dark"]`.

```css
.dark {
  --altech-background: #090f1d;
  --altech-foreground: #edf3ff;
  --altech-primary: #5b8dff;
  --altech-primary-foreground: #07102a;
  --altech-muted: #131e35;
  --altech-border: #263250;
}
```

## Recommendations

- Keep contrast high for text and surfaces.
- Avoid overly saturated accents in dark UI.
- Test focus ring visibility in both modes.
