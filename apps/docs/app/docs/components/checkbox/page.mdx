# Checkbox

Use Checkbox for multi-select boolean options.

## Quick Preview

<CheckboxDemo />

## Preset Examples

```tsx
<Checkbox variant="primary" size="sm" />
<Checkbox variant="primary" size="md" defaultChecked />
<Checkbox variant="success" size="lg" />
<Checkbox variant="danger" size="md" />
```

## Usage

```tsx
import { Checkbox } from "@altech-ui/react"

<Checkbox defaultChecked />
<Checkbox checked={checked} onCheckedChange={setChecked} />
<Checkbox variant="success" size="lg" />
```

## Props

- `variant?`: `primary | success | danger`
- `size?`: `sm | md | lg`
- Radix Checkbox `Root` props
