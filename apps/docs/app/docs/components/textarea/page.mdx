```tsx

<Textarea variant="filled" size="sm" placeholder="Quick note" />
```

<TextareaPlayground />

## Usage

```tsx
import { Textarea } from "@altech-ui/react"

<Textarea placeholder="Write your message" />
<Textarea variant="filled" />
<Textarea error="Message is required" />
```

## Props

- `error?`: `string`
- `variant?`: `default | filled | flushed | ghost`
- `size?`: `sm | md | lg`
- `fullWidth?`: `boolean`
- Native `textarea` props
