# Altech UI

_Introduction_

Altech UI is a React component library built with Tailwind CSS to help teams build fast, consistent, and scalable interfaces.

It is designed with TypeScript, a simple component API, and polished micro-interactions to make products feel modern.

## Documentation

- [Installation](/docs/installation)
- [Theming](/docs/theming)
- [Components](/docs/components)
- [GitHub Repository](https://github.com/Luxxn12/altech-ui)

## Tech Stack

Client: Next.js, React, TypeScript, Tailwind CSS, Nextra, MDX  
Library: `@altech-ui/react`, CVA, Framer Motion, Radix UI primitives
