# Altech UI

_Introduction_

Altech UI merupakan library komponen React berbasis Tailwind CSS yang dibuat untuk membantu tim membangun interface yang cepat, konsisten, dan mudah diskalakan.

Dirancang dengan TypeScript, component API yang sederhana, serta animasi micro-interaction yang halus agar produk terasa modern.

- [Installation](/docs/installation)
- [Theming](/docs/theming)

- [GitHub Repository](https://github.com/Luxxn12/altech-ui)

## Tech Stack

Client: Next.js, React, TypeScript, Tailwind CSS, Nextra, MDX  
Library: `@altech-ui/react`, CVA, Framer Motion, Radix UI primitives
