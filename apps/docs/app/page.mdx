# Altech UI

Build premium React interfaces with a component library designed for speed, consistency, and product-quality UX.

> Altech UI is built with **Tailwind CSS + TypeScript** and published as **`@altech-ui/react`**.

## Why Altech UI?

- Reusable React components with clean API
- Type-safe props and variant system (CVA)
- Accessible-first defaults
- Smooth micro-interactions and modern look
- First-class support for Next.js and Tailwind

## Package

```bash
npm i @altech-ui/react
```

## What you get today

- Stable: Button, Input, Card, Badge, Table, Modal
- Planned: Dropdown, Tabs, Alert, Avatar, Checkbox, Radio, Select, Textarea, Switch, Tooltip, layout primitives

Go to [Documentation](/docs) to start integrating components.
