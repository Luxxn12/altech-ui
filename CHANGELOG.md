# Changelog

## 0.1.0 - 2026-05-05

- Initial monorepo setup with pnpm workspace + Turborepo.
- Added `@altech-ui/react` package.
- Added core components: Button, Input, Card, Badge, Table, Modal.
- Added variant system with `class-variance-authority` and `cn()` utility.
- Added token-based theming and dark mode support.
- Added docs app (`apps/docs`) with component previews, code copy, sidebar, search, and dark mode toggle.
